Valores de RD Arquitecta

En el estudio tenemos una serie de preceptos que tomamos como valores innegociables a la hora de tomar y llevar a cabo cada proyecto.

Originalidad

Cada proyecto que ideamos es único, todo se realiza a medida con el cliente y partiendo de ideas originales.

Comunicación

Consideramos que para mantener armonía entre el trinomio de cada proyecto compuesto por cliente/estudio/obra debe haber una excelente comunicación manteniendo ese orden.

Como trabajamos

  • Entrevistas

    Nos reunimos con les clientes para que nos cuenten su idea, de la cual empezamos a esbozar los ejes principales del proyecto.

  • Reunion de equipo

    Con el resultado de la primera entrevista se arma junto con el equipo un plan de acción y diseño, donde se arman propuestas para les clientes.

  • Armado de equipo

    Una vez que se repitieron los dos pasos anteriores las veces necesarias como para que el proyecto se traslade al plano real, se contacta con el contratista, para que reúna la mano de obra necesaria, y se asesora sobre posibles proveedores de materiales.

  • Obra

    En conjunto con el contratista se procede a realizar el control de avance de obra junto con la dirección de la misma.
