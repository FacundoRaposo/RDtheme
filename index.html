Valores de RD Arquitecta

En el estudio tenemos una serie de preceptos que tomamos como valores inegociables a la hora de tomar y llevar a cabo cada proyecto.

Originalidad

Cada proyecto que ideamos es unico, todo se realiza a medida con el cliente y partiendo de ideas originales.

Comunicación

Consideramos que para mantener armonia entre el trinomio de cada proyecto compuesto por cliente/estudio/obra debe haber una excelente comunicacion manteniendo ese orden.

Paciencia

El tiempo para cada proyecto es distinto, hay multiples factores que afectan el desarrollo de cada trabajo por eso ante todo hay que ser pacientes para que el resultado final no se vea afectado

Como trabajamos

  • Entrevistas

    Nos reunimos con les clientes para que nos cuenten su idea, de la cual empezamos a esbozar los ejes principales del proyecto.

  • Reunion de equipo

    Con el resultado de la primer entrevista se arma junto con el equipo un plan de accion y diseño, donde se arman propuestas para les clientes.

  • Armado de equipo

    Una vez de que se repitieron los dos pasos anteriores las veces necesarias como para que el proyecto se traslade al plano real, se contacta con el contratista, para que reuna la mano de obra necesaria, y se asesora sobre posibles proveedores de materiales.

  • Obra

    En conjunto con el contratista se procede a realizar el control de avance de obra junto con la direccion de la misma.
